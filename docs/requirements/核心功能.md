# 核心功能

## 多模态对话式需求输入

### 输入形式
- **自然语言描述**：用户用文字描述动画需求（类型、风格、内容、时长等）
- **图片参考**：用户上传排版参考图、风格参考图、截图等
  - 支持格式：PNG、JPG、WEBP、GIF
  - 单张最大 10MB
  - 最多同时上传 3 张参考图
- **素材文件**：视频、音频、数据文件等可作为动画素材
- **多模态 LLM 解析视觉布局**：提取图片中的元素位置、层级、风格、配色

### 输入限制与约束

| 输入类型 | 限制条件 | 超限处理 |
|----------|----------|----------|
| 文字描述 | 单条 ≤ 5000 字符 | 截断并提示精简 |
| 参考图片 | 单张 ≤ 10MB，≤ 3 张 | 拒绝上传并提示 |
| 图片分辨率 | 建议 ≥ 100×100 | 低分辨率警告但仍可上传 |
| 数据文件 | CSV/JSON ≤ 10MB，XLSX ≤ 50MB | 拒绝并建议拆分 |

---

## 核心工作流：参考文稿 + 参考排版 → 动画生成

1. 用户输入参考文稿（文案/脚本/数据）
2. 用户上传参考动画排版图片
3. Agent 解析并自动生成 Remotion 动画
4. 实时视频预览
5. 对话微调，满意后导出

## 动画生成能力

### 重点动画类型

#### 1. 数据动画

| 子类型 | 输入参数 | Remotion 实现方式 | 复杂度 |
|--------|----------|-------------------|--------|
| **柱状图** | 数据文件、颜色方案、动画时长 | 自定义 ChartLayer + `useCurrentFrame` 驱动柱子高度 | 中 |
| **折线图** | 数据文件、颜色方案、是否填充面积 | 自定义 ChartLayer + SVG path 逐帧绘制 | 中 |
| **饼图/环形图** | 数据文件、颜色方案、是否显示标签 | 自定义 ChartLayer + SVG arc 角度动画 | 中 |
| **数字滚动** | 起始值、结束值、持续时间、前缀/后缀 | 自定义 NumberLayer + `interpolate()` | 低 |
| **进度条** | 目标百分比、颜色、是否显示百分比文字 | 自定义 ShapeLayer + width 动画 | 低 |
| **仪表盘** | 目标值、范围、颜色区间 | 自定义 ChartLayer + SVG arc | 高 |

#### 2. 图标动画

| 子类型 | 输入参数 | Remotion 实现方式 | 复杂度 |
|--------|----------|-------------------|--------|
| **图标入场** | 图标素材/SVG、入场类型、持续时间 | ImageLayer + 预设入场动画 | 低 |
| **图标变形** | 起始 SVG 路径、目标 SVG 路径 | SVG path morphing + `useCurrentFrame` | 高 |
| **路径动画** | SVG 路径、描边颜色、持续时间 | SVG `<path>` + `stroke-dasharray` 动画 | 中 |
| **图标组合** | 多个图标、组合方式、入场顺序 | 多个 ImageLayer + Sequence 错开 | 中 |

#### 3. 文字动画

| 子类型 | 输入参数 | Remotion 实现方式 | 复杂度 |
|--------|----------|-------------------|--------|
| **标题动画** | 文字内容、字体、颜色、入场类型 | TextLayer + 预设入场动画 | 低 |
| **打字机效果** | 文字内容、打字速度（字符/秒） | TextLayer + `useCurrentFrame` 控制显示字符数 | 低 |
| **字幕条** | 文字内容、背景色、出现/消失动画 | TextLayer + ShapeLayer（背景）组合 | 中 |
| **文字逐字入场** | 文字内容、每字延迟帧数、入场类型 | 拆分字符为多个 TextLayer + Sequence | 中 |
| **文字滚动** | 文字内容、滚动方向、速度 | TextLayer + `transform.position` 关键帧 | 低 |

### 其他动画类型

| 类型 | 说明 | 支持程度 |
|------|------|----------|
| **转场效果** | 片段间切换动画（淡入淡出、滑动、缩放等） | 片段级别入场/出场动画 |
| **粒子效果** | 星星、雪花、光点等粒子系统 | 基础粒子（数量 < 100），复杂粒子暂不原生支持 |
| **形状动画** | 矩形、圆形、线条的形状变化和路径动画 | 基础形状 + 关键帧动画 |
| **相机运动** | 画面平移、缩放、旋转模拟镜头运动 | 通过关键帧模拟（暂不支持 3D 镜头） |

### 暂不支持（需告知用户）

| 效果类型 | 不支持原因 | 替代方案 |
|----------|-----------|----------|
| 3D 模型动画 | Remotion 基于 2D React，无 3D 渲染能力 | 使用 2D 模拟 3D 效果的预设 |
| 流体/烟雾效果 | 需要复杂物理模拟和着色器 | 使用预渲染视频素材叠加 |
| 手写/涂鸦动画 | 需要 SVG 路径数据，自动生成困难 | 用户上传 SVG 路径素材 |
| 复杂粒子（> 500） | 性能限制 | 减少粒子数量或预渲染 |

## 官方预设库

### 预设分类

| 分类 | 预设示例 | 可调参数 |
|------|----------|----------|
| **数据图表** | 柱状图-商务蓝、折线图-科技绿、饼图-简约白 | 数据、颜色、动画速度、标签样式 |
| **标题动画** | 标题-大气入场、标题-弹性弹出、标题-打字机 | 文字内容、字体、颜色、持续时间 |
| **转场预设** | 淡入淡出、左滑入、缩放弹出、模糊过渡 | 持续时间、缓动函数 |
| **片头片尾** | 片头-Logo 飞入、片尾-订阅关注、片头-图文快闪 | Logo 素材、文字内容、配色 |
| **社交媒体** | 竖屏标题、方形片头、弹幕效果 | 文字内容、字体、背景 |

### 预设存储格式

预设以 JSON 文件形式存储，包含完整的 timeline 片段数据：

```json
{
  "id": "preset-title-fade-in",
  "name": "标题-淡入",
  "category": "title",
  "description": "标题从透明渐显，带轻微上移",
  "template": {
    "type": "text",
    "transform": {
      "position": { "x": 960, "y": 540 },
      "opacity": 0
    },
    "animations": {
      "in": { "type": "fade", "durationInFrames": 20 }
    },
    "keyframes": [
      { "property": "transform.position.y", "frame": 0, "value": 560, "easing": "ease-out" },
      { "property": "transform.position.y", "frame": 20, "value": 540, "easing": "ease-out" }
    ]
  }
}
```

### 预设使用流程

1. 用户在对话面板输入"使用'标题-淡入'预设"
2. Agent 加载对应预设 JSON
3. 将预设模板中的占位符（如文字内容）替换为用户输入
4. 生成完整的 clip 数据并添加到时间线

## 输出格式

### 视频文件
- **MP4 (H.264)**：默认格式，兼容性最好
- **WEBM**：可选格式，文件体积较小

### 可编辑工程
- 导出完整的 Remotion 项目文件夹（含源码和素材）
- 用户解压后执行 `npm install && npm run start` 即可独立运行
- 支持在浏览器中打开进行代码级修改
