# UI 布局与交互设计

> **实施状态（2026-06-30）**：**M0–M9 ✅**（Windows MVP 预发行 v0.1.2）；后续以功能增强为主。权威入口：仓库根 [README.md](../../README.md)、[开发里程碑与路线图.md](./开发里程碑与路线图.md)。

> **本版本**：v1.0 优化版（v0.3 旧版已废弃并归档至 git 历史，本文档为唯一 UI 设计基准）
> **优化目标**：融合 **Adobe Premiere Pro** 的专业性与 **剪映 (CapCut)** 的极简体验，降低剪辑师和无编程经验用户的上手门槛。
> **核心原则**：**预览优先、操作即达、智能辅助、减少选择**。
> **关联文档**：[设计系统 Master](../design-system/easymotion/MASTER.md)

## 〇、设计系统

> 本节为基础规范，所有布局和组件必须遵循。

### 0.1 颜色系统

采用 **shadcn Neutral Dark**（2026-06 更新，替代原 Cinema Dark + Action Red）。壳层为中性灰黑 + 浅灰白主按钮；删除/错误仍用 destructive 红。

| 角色 | 实现 | 用途 |
|------|------|------|
| **背景主色** | `--background` oklch(0.145 0 0) | 主窗口、面板 |
| **抬升面** | `--card` / `--popover` | 卡片、浮层 |
| **边框** | `--border` white 10% | 分隔线 |
| **文字主/次** | `--foreground` / `--muted-foreground` | 标题与说明 |
| **主按钮** | `--primary` 浅灰白 | 极少数 CTA（非满屏红） |
| **聚焦环** | `--ring` 中性灰 | 输入框、键盘焦点 |
| **警告/错误** | `--warning` / `--destructive` | 静音、删除等 |

权威 token：`apps/electron/src/renderer/src/index.css`；设计说明见 `docs/design-system/easymotion/MASTER.md`。

**暗色主题实现**：
- shadcn `.dark` on `<html>`（v1.0 仅 dark）
- `:root` 与 `.dark` 同步同一套 oklch 变量

### 0.2 字体系统

| 角色 | 字体 | 字重 | 用途 |
|------|------|------|------|
| **界面字体** | Plus Jakarta Sans | 400/500/600 | 所有 UI 文字（菜单、按钮、标签） |
| **等宽字体** | JetBrains Mono | 400 | 时间码、代码片段、文件名 |

字号层级：`text-xs`(12px) / `text-sm`(14px) / `text-base`(16px) / `text-lg`(18px) / `text-xl`(20px) / `text-2xl`(24px)。

### 0.3 间距与圆角

- 间距 Token：`space-1`(4px) / `space-2`(8px) / `space-3`(12px) / `space-4`(16px) / `space-6`(24px)
- 圆角 Token：`rounded-sm`(4px，按钮/标签) / `rounded-md`(6px，输入框/卡片) / `rounded-lg`(8px，面板/弹窗)

### 0.4 组件规范

- **按钮**（shadcn `Button`，radix-nova）：
  - 工具栏图标：`variant="ghost" size="icon"`
  - 常规操作：`variant="outline"` 或 `secondary`
  - 高强调（稀少）：`variant="default"`（浅灰白实心）
  - 危险：`variant="destructive"` + 必要时 `AlertDialog`
- **输入框**：shadcn `Input` / `Textarea`；聚焦 `ring-ring`
- **面板/卡片**：`bg-background border-border rounded-lg`

### 0.5 动画规范

| 场景 | 时长 | 缓动函数 |
|------|------|----------|
| 按钮悬停（颜色变化，**不位移**） | 150ms | `ease-out` |
| 面板展开/收起 | 200ms | `cubic-bezier(0.4, 0, 0.2, 1)` |
| 弹窗出现 | 150ms | `ease-out` + `scale(0.95→1)` |
| 时间线播放头移动 | 实时 | 无过渡，精确到帧 |
| 轨道拖拽 | 100ms | `ease-out` |

**性能要求**：所有动画使用 `transform` 和 `opacity`（避免触发重排）；支持 `prefers-reduced-motion` 媒体查询。

### 0.6 反模式清单（Do & Don't）

| 规则 | Do | Don't |
|------|-----|-------|
| **图标** | 使用 Lucide 图标（24x24 viewBox） | 使用 Emoji 作为图标 |
| **光标** | 所有可点击元素加 `cursor-pointer` | 可点击区域无指针反馈 |
| **悬停** | 颜色/透明度变化（不位移） | 缩放导致布局偏移 |
| **对比度** | 文字对比度 ≥ 4.5:1 | 灰色文字在深色背景上过浅 |
| **边框** | 使用 `border-[#2D2D4A]` | 使用接近背景色的边框 |
| **玻璃效果** | 不透明度 ≥ 80% | 过透明导致内容看不清 |

## 一、设计哲学：专业 + 极简

### 1.1 为什么参考 Premiere + 剪映

| 参考对象 | 核心优点 | 我们要吸收什么 |
|----------|----------|--------------|
| **Premiere Pro** | 强大的时间线编辑、轨道管理、快捷键体系、吸附精度 | 三栏布局骨架、轨道控制逻辑、专业快捷键、吸附系统 |
| **剪映 (CapCut)** | 界面清爽、一键操作、智能辅助、拖拽体验极佳、学习成本极低 | "选中即操作"的浮动工具条、空状态引导、素材拖拽即所得、智能对齐 |

### 1.2 "方便、简单"的量化标准

- **首次使用**：新用户在 3 分钟内完成第一个动画生成（原目标是 5 分钟）
- **零学习成本**：所有核心操作不依赖右键菜单，主界面可见即可操作
- **减少点击**：80% 的常用操作可以在 2 次点击内完成
- **智能默认**：AI 生成结果即"可用"，不需要用户手动调整大量参数

## 二、布局架构（核心变更）

### 2.1 旧版问题

旧版将**对话面板**放在中间工作区下方，导致：
- 预览窗口被严重挤压（视频编辑中预览是最重要的视觉区域）
- 时间线区域也被压缩
- 对话面板常驻占用空间，但用户并非一直在对话

### 2.2 新版布局：预览优先

```
┌──────────────────────────────────────────────────────────────────────┐
│                        顶部工具栏（精简）                             │
│  [≡] [撤销] [重做] [保存]                    [🤖 AI助手] [渲染] [导出] │
├──────────┬──────────────────────────────┬──────────────────────────┤
│          │                              │                          │
│  左侧面板 │        中间预览区域            │      右侧面板           │
│          │       （最大化占比）            │    （上下文自适应）      │
│  项目     │                              │                          │
│  素材     │   ┌──────────────────────┐   │   [属性 / 模板]          │
│  预设     │   │                      │   │                          │
│          │   │   Remotion 预览       │   │   [AI 助手] ← 新增       │
│          │   │   (iframe Player)    │   │                          │
│          │   │                      │   │                          │
│          │   └──────────────────────┘   │                          │
│          │                              │                          │
├──────────┴──────────────────────────────┴──────────────────────────┤
│                        底部时间线（充足空间）                         │
│  ⏮ ⏪ ⏸ ▶️ ⏩ ⏭  00:00:05 / 00:00:10    [🧲] [🔁] [▓▓░░▓▓] [1s▼]  │
│  ─────────────────────────────────────────────────────────────────  │
│  [轨道1] ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│  [轨道2] ━━◆━━━━━━━━◆━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
└──────────────────────────────────────────────────────────────────────┘
```

### 2.3 核心布局变更点

| 变更项 | 旧版 | 新版 | 理由 |
|--------|------|------|------|
| **对话面板位置** | 中间工作区下方（常驻） | **右侧面板第 4 个 Tab** 或 **底部可完全收起** | 释放垂直空间给预览和时间线；AI 作为"工具"而非"面板" |
| **顶部菜单栏** | 传统菜单栏 `[文件][编辑][视图]` | **精简工具栏**：只保留最常用操作 | 剪映风格：菜单项沉入汉堡菜单 `≡`，主界面只放高频按钮 |
| **预览区域占比** | 约 35% 垂直空间 | **约 55% 垂直空间** | 视频编辑的核心视觉区域必须最大 |
| **右侧面板** | 属性 / 快捷素材 切换 | **属性 / 模板**（AI 为独立右栏） | 避免与左侧素材库重复；收藏/最近使用并入左栏素材 Tab |

### 2.4 面板尺寸与响应式

| 面板 | 默认 | 最小 | 最大 | 可调整 | 收起后 |
|------|------|------|------|--------|--------|
| 左侧 | 220px | 160px | 350px | ✅ | 40px 图标栏 |
| 右侧 | 280px | 200px | 400px | ✅ | 完全隐藏 |
| 底部时间线 | 220px | 150px | 350px | ✅ | 40px 轨道标签栏 |
| 对话面板（右侧面板模式） | 280px（与右侧面板共享） | — | — | — | 切换 Tab |
| 对话面板（底部模式） | 200px | 48px（仅输入条） | 300px | ✅ | 48px 输入条 |

**响应式策略**：
- 窗口宽度 < 1400px：对话面板强制切换为底部 48px 模式，右侧面板优先保留属性 Tab
- 窗口宽度 < 1200px：提示"窗口过小，建议最大化"

### 2.5 当前实现（2026-06-27）

以下描述 **新 UI**（`apps/electron/src/renderer/src/`，非 Legacy）的已落地布局，与 §2.2 设计稿相比的主要差异：**AI 助手为独立右栏通高**；**预设库在左栏 Tab**；**预览列按 16:9 定宽**，横向余量分给左栏与属性栏；**右侧无独立素材 Tab**（收藏/最近使用并入左侧 `AssetsPanel`）。

```
┌ TitleBar（32px，自定义窗口壳 + 系统/自绘窗口控制）──────────────────┐
├ Frame（左右下边框）────────────────────────────────────────────────┤
│ TopToolbar：[≡ 应用菜单] [撤销][重做] … [AI][导出]                    │
├────────────────────────────────────────────┬───────────────────────┤
│ 左栏（项目/素材/预设）│ 预览（16:9 定宽）│ 属性 │ 时间线（仅本区宽度）│ AI 助手 │
│  flex 均分余宽       │ usePreviewColumn │ flex │ 默认高 480px      │ 通高   │
│  预设 Tab：scrollbar-theme 纵滚         │scrollbar│ 关键帧轨道区同  │ 可收起 │
└────────────────────────────────────────────┴───────────────────────┘
```

| 项 | 实现要点 |
|----|----------|
| **窗口壳** | `window-chrome.js`：`frame: false` + `thickFrame`（Win/Linux）；macOS `titleBarStyle: hiddenInset`；`TitleBar` + `WindowControls` |
| **应用菜单 `≡`** | `AppMenu.tsx`：文件（新建/打开/**关闭项目**/保存/导出/退出）、编辑、视图、帮助 |
| **关闭项目** | `main:project:close` → 停止预览、清空时间线/素材/对话，回到未打开项目状态；有未保存更改时确认 |
| **AI 助手** | `AiPanel.tsx` 独立列，**不**与时间线同宽；工具栏 AI 按钮可展开已收起的 AI 栏 |
| **预览宽度** | `usePreviewColumnWidth`：按行高与 16:9 计算列宽；左栏与属性栏 `flex: 1 1 0` 均分剩余横向空间（拖拽后可锁定像素宽） |
| **时间线高度** | `uiStore.timelineHeight` 默认 `480px`（`TIMELINE_HEIGHT_MAX`），范围 160–480px |
| **滚动条** | 预设库、右侧属性栏、时间线关键帧轨道用 `scrollbar-theme`；左侧「项目/素材」Tab 仍用 `ScrollArea` |
| **素材库** | `AssetsPanel`：导入/搜索/类型筛选；「全部」视图顶部展示收藏与最近使用（原右侧 `QuickAssetPanel` 已移除） |
| **预设缩略图** | 动图 WebP **不纳入 Git**；克隆后 `pnpm generate:preset-thumbnails`（见 [开发环境搭建指南.md](./开发环境搭建指南.md)） |

## 三、时间线交互优化（融合 Premiere 精度 + 剪映简洁）

### 3.1 轨道头部简化

**旧版问题**：轨道头部按钮太多（展开/折叠、锁定、可见、静音、名称、图标、添加），每行都很拥挤。

**新版设计**：

```
旧版（拥挤）:
│▼│🔒│👁️│🔇│ [轨道1 文字]  ━━━[片段A]━━━━━━━[片段B]━━━ │

新版（简洁）:
│ ▼ [👁 文字标题]  🔒  ━━━[片段A]━━━━━━━[片段B]━━━ │
```

| 元素 | 旧版 | 新版 | 说明 |
|------|------|------|------|
| **展开/折叠** | 独立箭头 ▼ | **双击轨道名称** 或 小箭头 | 减少按钮数量 |
| **可见性** | 眼睛图标 👁️ | **轨道名称前的类型图标**（点击切换可见/变灰） | 把状态融入图标 |
| **锁定** | 锁图标 🔒 | **锁定后轨道变灰 + 名称旁小锁**，未锁定不显示 | 默认隐藏，减少视觉噪音 |
| **静音** | 喇叭图标 🔇 | **仅音频轨道显示，其他轨道不显示该位置** | 按轨道类型动态显示 |
| **轨道名称** | 可双击重命名 | **保持可双击重命名**，增加右键菜单 | 不变 |
| **添加片段** | ➕ 按钮 | **移除固定按钮**，改为**右键轨道**或**顶部工具栏** | 减少每行固定按钮 |

**轨道颜色标记**（新增，参考 Premiere）：
- 右键轨道 → "设置颜色" → 轨道左侧显示 4px 彩色竖条
- 帮助用户快速区分轨道类型和用途

### 3.2 选中片段：浮动快捷工具条（剪映核心体验）

**这是最重要的交互优化之一。**

当用户**单击选中时间线上的片段**时，在片段**上方或时间线顶部**弹出浮动工具条：

```
                    ┌─────────────────────────────────────┐
    [片段块] ───────│  ✂️ 分割  │  🗑 删除  │  📋 复制  │  🎬 动画  │  ⏱ 速度  │
                    └─────────────────────────────────────┘
```

| 按钮 | 功能 | 说明 |
|------|------|------|
| **✂️ 分割** | 在当前播放头位置分割片段 | 替代右键"拆分"，最常用操作前置 |
| **🗑 删除** | 删除选中片段 | 替代按 Delete 键，对鼠标用户更友好 |
| **📋 复制** | 复制片段（可粘贴到同轨道或其他轨道） | |
| **🎬 动画** | 快速设置入场/出场动画 | 弹出小面板：淡入/滑入/缩放/打字机 |
| **⏱ 速度** | 调整片段时长（拉伸/压缩） | 替代拖拽边缘 |
| **🎨 样式** | 快速修改颜色/字体 | 仅文字片段显示 |

**交互细节**：
- 工具条出现动画：200ms `ease-out`，从片段中心向上淡入 + 位移 8px
- 工具条自动隐藏：点击空白处或选中其他片段时，工具条 150ms 淡出
- 工具条位置：优先显示在片段上方；如果上方空间不足（片段靠近时间线顶部），则显示在片段下方
- **快捷键仍然保留**：空格播放、Delete 删除、Ctrl+Z 撤销等，工具条只是为鼠标用户提供快捷入口

### 3.3 智能对齐与吸附（新增，参考 Figma + 剪映）

**旧版吸附**：片段边界吸附 + 播放头吸附。

**新版智能对齐**：

当用户**拖拽片段**时，如果片段的边缘/中心与其他片段或画布对齐，显示**智能对齐线**：

```
      │                  │
      │  [片段A]         │  [片段B]
      │                  │
──────┼──────────────────┼──────  ← 水平对齐线（绿色虚线）
      │                  │
      │     [片段C]      │
      │                  │
```

| 对齐类型 | 触发条件 | 视觉反馈 |
|----------|----------|----------|
| **左边缘对齐** | 片段左边缘与其他片段左边缘 X 坐标差 < 10px | 垂直绿色虚线 |
| **右边缘对齐** | 片段右边缘与其他片段右边缘 X 坐标差 < 10px | 垂直绿色虚线 |
| **中心对齐** | 片段中心与其他片段中心 X 坐标差 < 10px | 垂直绿色虚线 + "中心"提示 |
| **等距分布** | 三个及以上片段间距相等 | 间距标注 "= 30帧" |
| **画布居中** | 片段中心与画布中心差 < 10px | 十字对齐线 + "居中"提示 |
| **垂直位置对齐** | 跨轨道的片段 Y 位置对齐（预览中的垂直位置） | 水平绿色虚线 |

**吸附开关**：
- 默认开启，工具栏显示 🧲 图标（高亮=开启，灰色=关闭）
- 拖拽时按住 `Alt` 临时禁用吸附（与旧版一致）
- 增加"弱吸附"：对齐线显示但不强制吸附，用户可继续微调（Figma 风格）

### 3.4 一键对齐按钮（新增，剪映风格）

在时间线工具栏或右侧面板顶部，增加**对齐工具组**：

```
[ 水平居中 ] [ 垂直居中 ] [ 左对齐 ] [ 右对齐 ] [ 等距分布 ]
```

- 选中 1 个片段 → "水平居中"将片段在画布中水平居中
- 选中 2+ 片段 → "水平居中"将片段在彼此间水平均匀分布
- 所有对齐操作自动添加关键帧（如果当前播放头位置没有关键帧），确保对齐效果可预览

### 3.5 片段交互手势增强

| 手势 | 作用 | 行为 |
|------|------|------|
| **单击片段** | 选中 | 片段高亮 + 右侧面板显示属性 + **弹出浮动工具条** |
| **双击片段** | 快速定位 | 播放头跳到片段起始帧 + 预览窗口 seek |
| **拖拽片段主体** | 移动位置 | 修改 `startInFrames`，显示智能对齐线 |
| **拖拽片段左边缘** | 裁剪开头 | 修改 `startInFrames` 和 `durationInFrames`（Ripple Edit 模式） |
| **拖拽片段右边缘** | 裁剪结尾 | 修改 `durationInFrames` |
| **拖拽片段 + Shift** | 复制并移动 | 创建副本，拖拽的是新副本 |
| **滚轮** | 水平滚动 | 时间线左右滚动 |
| **Shift + 滚轮** | 缩放时间线 | 缩放时间轴比例 |
| **Ctrl + 滚轮** | 垂直滚动 | 轨道列表上下滚动 |
| **中键拖拽** | 平移时间线 | 类似 Premiere 的手形工具 |
| **右键片段** | 上下文菜单 | 完整菜单（分割/复制/删除/复制样式/颜色标记/属性） |

## 四、对话面板重构：从常驻到召唤式

### 4.1 问题分析

EasyMotion 的核心卖点是"对话式生成"，但用户**并非 100% 时间都在对话**。将对话面板常驻会：
- 挤压预览区域
- 让界面显得臃肿
- 与剪映的极简风格冲突

### 4.2 方案：双模式对话面板

#### 模式 A：右侧面板 Tab（推荐用于宽屏）

右侧面板增加第 4 个 Tab：

```
┌────────────────────────────────┐
│  [属性] [素材] [预设] [🤖 AI助手] │
├────────────────────────────────┤
│                                │
│  🤖 AI 正在分析图片...          │
│                                │
│  ┌────────────────────────┐   │
│  │ 用户：做一个标题         │   │
│  └────────────────────────┘   │
│  ┌────────────────────────┐   │
│  │ AI：已为您生成淡入标题   │   │
│  │ [✅ 应用] [🔄 重新生成]  │   │
│  └────────────────────────┘   │
│                                │
│  [描述你的动画...      ] [➤]  │
└────────────────────────────────┘
```

**特点**：
- 与属性面板同层级，用户可以在"调整属性"和"对话调整"之间快速切换
- AI 生成结果提供 **"应用" / "重新生成"** 两个明确按钮，减少用户犹豫
- 右侧面板宽度足够显示完整对话历史（约 280px）

#### 模式 B：底部可收起条（适用于窄屏或用户偏好）

```
┌──────────────────────────────────────────────────────────────┐
│                        预览窗口                               │
├──────────────────────────────────────────────────────────────┤
│                        时间线                                 │
├──────────────────────────────────────────────────────────────┤
│ 🤖 [描述你的动画需求，或拖拽图片到此处...     ] [➤] [🖼️] │  ← 收起态（48px）
└──────────────────────────────────────────────────────────────┘
```

**收起态**：
- 只显示一行输入框 + AI 头像 + 图片上传 + 发送按钮
- 高度 48px，不占用空间
- 点击输入框或 `▲` 按钮 → **展开为完整对话面板**（高度 200-300px，覆盖时间线上方）

**展开态**：
- 从底部向上展开，覆盖时间线区域（时间线被压缩或隐藏）
- 显示完整对话历史
- 再次点击 `▼` 收起

**模式切换**：用户可以在设置中选择偏好模式。

### 4.3 AI 消息快捷操作（新增）

每条 AI 回复消息下方增加**快捷操作按钮**：

```
┌──────────────────────────────────┐
│ AI：已将标题字体调大至 96px      │
│ [👁 预览效果] [✅ 确认] [🔄 撤销] │
└──────────────────────────────────┘
```

| 按钮 | 作用 |
|------|------|
| **👁 预览效果** | 播放头跳到该片段位置，自动播放 2 秒预览 |
| **✅ 确认** | 确认应用本次修改，消息变为"已确认"状态（绿色边框） |
| **🔄 撤销** | 撤销本次 AI 修改，恢复到修改前状态 |
| **💬 继续调整** | 输入框自动填充 "在此基础上，再..."，方便连续对话 |

**目的**：降低用户的心理负担——AI 修改后用户不需要"害怕"，可以一键预览、确认或撤销。

### 5.3 "常用调整"智能区（新增）

**原理**：根据用户选中片段的类型，把**最常用的 3-5 个属性**置顶显示。

| 片段类型 | 常用调整（置顶显示） |
|----------|----------------------|
| **文字** | 字体大小、颜色、位置 X/Y |
| **图片** | 缩放、位置 X/Y、填充模式 |
| **视频** | 音量、播放速度、裁剪起点/终点 |
| **形状** | 填充色、描边色、圆角、尺寸 |
| **图表** | 数据文件、主题色、动画速度 |

**交互**：
- 用户修改某个属性后，该属性会被记录为"常用"，下次选中同类型片段时置顶
- 用户可手动 pin/unpin 属性到常用区
- 常用区最多显示 5 个属性，超出后最早使用的被移出

### 5.4 实时预览优化

| 控件类型 | 旧版行为 | 新版行为 | 理由 |
|----------|----------|----------|------|
| **数值输入框** | 失去焦点或按 Enter 触发 | **拖拽输入框旁的小滑块**实时更新 + Enter 确认 | 像剪映一样，数值可以"搓" |
| **颜色选择器** | 选择后触发 Generator | **选择后立即更新 previewStore**，延迟 300ms 触发 Generator | 颜色需要即时视觉反馈 |
| **滑块** | 拖拽结束触发 | **拖拽过程中实时更新 preview**，拖拽结束触发 Generator | 即时反馈 |
| **下拉框** | 选择后立即触发 | 保持不变 | 合理 |

## 六、预览窗口优化

### 6.1 最大化视觉区域

- 预览窗口独占中间工作区的全部可用空间
- 默认显示**画布安全框**（可关闭）：9:16 区域提示线（适配竖屏视频）
- 默认显示**中心十字辅助线**（可关闭）：帮助用户对齐元素

### 6.2 点击选中交互（新增）

当用户**点击 iframe 内的元素**时：

```
┌──────────────────────────────┐
│        ┌──────────┐         │
│        │ Hello    │  ← 点击 │
│        │ [红色边框高亮]│     │
│        └──────────┘         │
│              ↑              │
│    iframe 内元素被红色边框    │
│    高亮，同时时间线选中对应片段│
└──────────────────────────────┘
```

**交互流程**：
1. 用户点击 iframe 内的文字 "Hello"
2. iframe 内该文字元素显示**红色边框高亮**（通过 iframe postMessage 通知 Remotion Player 添加高亮 overlay）
3. 父窗口时间线自动选中对应片段
4. 右侧面板自动切换到属性面板，显示该文字片段的属性
5. 播放头自动跳到该片段的起始帧

**空状态**：
- 无子项目：显示项目分辨率虚线框 + "点击 🤖 AI助手 或导入素材开始"
- Generator 生成中：半透明遮罩 + 旋转 loading + "正在生成预览..."
- 生成失败：错误图标 + "生成失败" + [查看错误] [重新生成]

### 6.3 预览工具栏精简

```
旧版:  ◀ ▶ ⏸  00:00:00 / 00:00:10   [100%] 🔍
新版:  ⏮ ⏪ ⏸ ▶️ ⏩ ⏭  150/300帧  00:05:00  [ fit ▼ ]
```

**变更**：
- 按钮按 Premiere 标准排列：跳到开头、上一帧、播放/暂停、下一帧、跳到结尾
- 时间码同时显示**帧号**和**时间**（`150/300帧  00:05:00`）
- 缩放控制改为下拉框：`fit` / `25%` / `50%` / `100%` / `200%`
- **移除全屏按钮**（改为双击预览窗口进入全屏，像剪映一样）

### 6.4 性能降级提示（新增）

当预览帧率持续低于 15fps 超过 3 秒时，在预览窗口**右上角**显示一个小提示：

```
┌──────────────────────────────┐
│  ⚡ 预览帧率较低  [降低质量]  │
│                              │
│        ...预览内容...         │
└──────────────────────────────┘
```

- 点击"降低质量" → 自动将预览质量从 `full` 降到 `half`
- 提示 3 秒后自动淡出
- 用户也可以在设置中手动选择预览质量

| Toast 提示 | 200ms | ease-out | 从右向左滑入，3 秒后淡出 |

| 复制片段 | `Ctrl+D` / `Cmd+D` | 复制选中片段到后方 |
| 快速添加文字 | `T` | 按 T 后点击时间线，在该位置添加文字片段 |
| 快速添加图片 | `I` | 按 I 后选择图片素材，点击时间线放置 |
| 取消选中 | `Esc` | 全局 |
| 左侧面板开关 | `Ctrl+Shift+L` | 快速释放空间 |
| 右侧面板开关 | `Ctrl+Shift+R` | 快速释放空间 |
| 时间线开关 | `Ctrl+Shift+T` | 全屏预览模式 |

### 10.3 提示策略

- **Tooltip**：鼠标悬停按钮 1.5 秒后显示 tooltip，包含快捷键提示（如 "播放/暂停 (Space)"）
- **快捷键面板**：`Ctrl+/` 或 `Cmd+/` 打开快捷键速查面板
- **新手提示**：首次使用时，关键操作（分割、播放）显示一次浮动提示
