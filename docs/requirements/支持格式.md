# 支持格式

## 输入素材

### 图片

| 格式 | 扩展名 | 最大分辨率 | 颜色模式 | 透明度 | 说明 |
|------|--------|------------|----------|--------|------|
| PNG | `.png` | 无限制 | RGB/RGBA | 支持 | 推荐用于 Logo、图标等需要透明的素材 |
| JPG/JPEG | `.jpg`, `.jpeg` | 无限制 | RGB | 不支持 | 推荐用于照片、背景图 |
| WEBP | `.webp` | 无限制 | RGB/RGBA | 支持 | 体积更小，推荐优先使用 |
| SVG | `.svg` | 矢量，无分辨率限制 | - | 支持 | 图标、Logo 最佳格式，导入时转为 PNG 预览 |
| GIF | `.gif` | 建议 ≤ 1920×1080 | 索引色 | 支持（1-bit） | 动图通过 `@remotion/gif` 组件直接渲染 |

### 视频

| 格式 | 扩展名 | 推荐编码 | 最大时长 | 最大文件大小 | 说明 |
|------|--------|----------|----------|--------------|------|
| MP4 | `.mp4` | H.264 (AVC) | 无限制 | 2GB | 最推荐格式，兼容性最好 |
| MOV | `.mov` | H.264 (AVC) / ProRes | 无限制 | 2GB | Apple 设备常用格式 |
| WEBM | `.webm` | VP8/VP9 | 无限制 | 2GB | 体积较小，适合网络传输 |

**视频素材处理：**
- 导入时自动提取首帧作为缩略图（使用 FFmpeg）
- 导入时读取视频元信息（分辨率、时长、帧率）存入 SQLite
- 视频素材在 Remotion 中通过 `<Video>` 或 `<OffthreadVideo>` 组件渲染

### 音频

| 格式 | 扩展名 | 推荐采样率 | 声道 | 最大文件大小 | 说明 |
|------|--------|------------|------|--------------|------|
| MP3 | `.mp3` | 44.1kHz / 48kHz | 单声道/立体声 | 500MB | 最常用格式 |
| WAV | `.wav` | 44.1kHz / 48kHz | 单声道/立体声 | 500MB | 无损格式，体积较大 |
| AAC | `.aac`, `.m4a` | 44.1kHz / 48kHz | 单声道/立体声 | 500MB | 高质量压缩 |

**音频素材处理：**
- 导入时生成音频波形缩略图
- 音频素材在 Remotion 中通过 `<Audio>` 组件渲染
- 支持音量控制（0-1）

### 数据文件

| 格式 | 扩展名 | 最大文件大小 | 编码 | 说明 |
|------|--------|--------------|------|------|
| CSV | `.csv` | 10MB | UTF-8（推荐）/ GBK（兼容） | 第一行为表头，支持标准 CSV 格式 |
| JSON | `.json` | 10MB | UTF-8 | 标准 JSON 数组格式，每个元素为一行数据 |
| XLSX | `.xlsx` | 50MB | - | Excel 2007+ 格式，自动读取第一个工作表 |

**数据文件处理：**
- 导入后自动解析为结构化数据，存入 `data/` 目录
- CSV 解析：检测分隔符（逗号/分号/制表符），检测编码
- XLSX 解析：读取第一个工作表，自动识别表头
- 导入时展示前 5 行预览供用户确认
- 数据文件不自动创建轨道，需在 ChartLayer 中手动绑定

### 字体

| 格式 | 扩展名 | 说明 |
|------|--------|------|
| TTF | `.ttf` | TrueType 字体 |
| OTF | `.otf` | OpenType 字体 |

**字体处理：**
- 导入时复制到项目 `fonts/` 目录
- 在 Remotion 组件中通过 `@font-face` CSS 规则引用
- 不安装到系统字体（保持项目可移植性）

## 输出格式

### 视频文件

| 格式 | 编码 | 推荐场景 | 说明 |
|------|------|----------|------|
| MP4 | H.264 (AVC) | 默认输出，通用兼容 | 支持硬件加速编码，质量与速度平衡 |
| WEBM | VP9 | 网页使用、体积敏感 | 相同质量下体积比 MP4 小 20-30% |

**导出参数：**

| 参数 | 选项 | 说明 |
|------|------|------|
| 分辨率 | 原始 / 1080p / 720p / 480p | 缩放导出时保持宽高比 |
| 质量 | 高 (CRF 18) / 中 (CRF 23) / 低 (CRF 28) | CRF 值越小质量越高，文件越大 |
| 帧率 | 原始 / 30fps / 24fps / 60fps | 可降帧率导出以减小体积 |

### 可编辑工程

- 导出完整的 Remotion 项目文件夹（含源码和素材）
- 打包为 ZIP 格式
- 包含：`package.json`、`remotion.config.ts`、`tsconfig.json`、`src/` 目录、`assets/` 目录
- 用户解压后执行 `npm install && npm run start` 即可在浏览器独立运行

## 格式不支持的提示

当用户尝试导入不支持的格式时：
- 拒绝导入并显示明确的错误提示："不支持的文件格式：.xxx。支持的格式：PNG, JPG, WEBP, SVG, GIF, MP4, MOV, WEBM, MP3, WAV, AAC, CSV, JSON, XLSX, TTF, OTF"
- 提供转换建议（如"您可以使用 FFmpeg 将 .avi 转换为 .mp4 后重试"）
