# UI 布局设计

## 设计系统

基于 `ui-ux-pro-max` 技能生成的专业暗色主题设计系统，适配视频编辑软件场景。

### 颜色系统

采用 **Cinema Dark + Action Red** 配色方案，确保长时间编辑的视觉舒适度。

| 角色 | 颜色值 | 用途 |
|------|--------|------|
| **背景主色** | `#0F0F23` | 主窗口背景、面板底色 |
| **背景次色** | `#1A1A2E` | 次级面板、输入框背景 |
| **背景三级** | `#252542` | 悬停状态、列表项交替 |
| **边框/分隔线** | `#2D2D4A` | 面板边框、分割线 |
| **文字主色** | `#F8FAFC` | 标题、主要文字 |
| **文字次色** | `#94A3B8` | 描述、次要文字、placeholder |
| **强调色（主）** | `#E11D48` | 播放头、关键帧、选中状态、CTA 按钮 |
| **强调色（次）** | `#0D9488` | 成功状态、进度条、激活指示 |
| **警告色** | `#F97316` | 警告提示、需要关注的状态 |
| **错误色** | `#DC2626` | 错误提示、删除操作 |

**暗色主题实现**：
- 使用 shadcn/ui 的 `.dark` 类方案
- CSS 变量定义：`--background: 240 10% 3.9%`
- 所有自定义组件必须同时支持 `:root` 和 `.dark` 两套色值

### 字体系统

| 角色 | 字体 | 字重 | 用途 |
|------|------|------|------|
| **界面字体** | Plus Jakarta Sans | 400/500/600 | 所有 UI 文字（菜单、按钮、标签） |
| **等宽字体** | JetBrains Mono | 400 | 时间码、代码片段、文件名 |

**字号层级**：
- `text-xs` (12px)：辅助信息、时间刻度
- `text-sm` (14px)：按钮文字、列表项、输入框
- `text-base` (16px)：正文、菜单项
- `text-lg` (18px)：面板标题
- `text-xl` (20px)：窗口标题
- `text-2xl` (24px)：大标题、空状态提示

### 间距与圆角

| Token | 值 | 用途 |
|-------|-----|------|
| `space-1` | 4px | 图标与文字间距 |
| `space-2` | 8px | 按钮内边距（垂直） |
| `space-3` | 12px | 列表项间距 |
| `space-4` | 16px | 面板内边距 |
| `space-6` | 24px | 区域间距 |
| `rounded-sm` | 4px | 按钮、标签 |
| `rounded-md` | 6px | 输入框、卡片 |
| `rounded-lg` | 8px | 面板、弹窗 |

### 组件规范

**按钮**：
- 主按钮：`bg-[#E11D48] text-white hover:bg-[#BE123C] rounded-sm px-4 py-2`
- 次按钮：`bg-[#252542] text-[#F8FAFC] hover:bg-[#2D2D4A] rounded-sm px-4 py-2 border border-[#2D2D4A]`
- 图标按钮：`p-2 rounded-sm hover:bg-[#252542]`
- 危险操作：`bg-[#DC2626] hover:bg-[#B91C1C]`

**输入框**：
- `bg-[#1A1A2E] border border-[#2D2D4A] rounded-md text-[#F8FAFC] placeholder:text-[#64748B]`
- 聚焦状态：`border-[#0D9488] ring-1 ring-[#0D9488]`

**面板/卡片**：
- `bg-[#0F0F23] border border-[#2D2D4A] rounded-lg`
- 悬浮阴影：`shadow-lg shadow-black/20`

### 动画规范

| 场景 | 时长 | 缓动函数 |
|------|------|----------|
| 按钮悬停 | 150ms | `ease-out` |
| 面板展开/收起 | 200ms | `cubic-bezier(0.4, 0, 0.2, 1)` |
| 弹窗出现 | 150ms | `ease-out` + `scale(0.95→1)` |
| 时间线播放头移动 | 实时 | 无过渡，精确到帧 |
| 轨道拖拽 | 100ms | `ease-out` |

**性能要求**：
- 所有动画使用 `transform` 和 `opacity`（避免触发重排）
- 支持 `prefers-reduced-motion` 媒体查询

### 反模式清单（Do & Don't）

| 规则 | Do | Don't |
|------|-----|-------|
| **图标** | 使用 Lucide 图标（24x24 viewBox） | 使用 Emoji 作为图标 |
| **光标** | 所有可点击元素加 `cursor-pointer` | 可点击区域无指针反馈 |
| **悬停** | 颜色/透明度变化（不位移） | 缩放导致布局偏移 |
| **对比度** | 文字对比度 ≥ 4.5:1 | 灰色文字在深色背景上过浅 |
| **边框** | 使用 `border-[#2D2D4A]` | 使用接近背景色的边框 |
| **玻璃效果** | 不透明度 ≥ 80% | 过透明导致内容看不清 |

## 各区域功能

### 顶部菜单栏
- **文件**：新建项目、打开项目、保存、导入素材
- **编辑**：撤销、重做、复制、粘贴、删除
- **视图**：面板显示/隐藏、全屏预览、布局切换
- **帮助**：文档、快捷键、关于
- **渲染**：渲染当前子项目
- **导出**：导出视频/工程文件

### 左侧面板（项目导航 + 素材库 + 预设库）

**定位**：左侧面板是**全局资源管理器**，管理整个项目的结构、素材和预设。

**布局结构：**
```
┌─────────────────────────────┐
│  [项目] [素材] [预设]        │  ← Tab 切换栏
├─────────────────────────────┤
│                             │
│   Tab 内容区域               │
│                             │
│                             │
└─────────────────────────────┘
```

**Tab 切换栏：**
- 三个 Tab："项目" / "素材" / "预设"
- 当前激活 Tab 带下划线高亮
- 支持快捷键切换：`Ctrl+1` / `Ctrl+2` / `Ctrl+3`
- 未选中时 hover 显示浅色背景

#### 项目 Tab

- **项目树**：显示总项目 → 子项目层级结构
  - 支持展开/折叠子项目（点击左侧箭头或双击）
  - 当前激活子项目高亮显示（背景色加深 + 左侧竖条指示器）
  - 点击子项目切换当前工作区
  - 右键子项目：重命名 / 复制 / 删除 / 导出
  - **空状态**：无子项目时显示"暂无子项目，点击 + 新建"

#### 素材 Tab

- **素材库**（主素材库）：
  - 顶部搜索框（placeholder："搜索素材..."）+ 类型筛选下拉框（全部/图片/视频/音频/数据）
  - 显示**当前总项目**下所有已导入素材
  - 按类型分组（图片、视频、音频、数据文件）
  - 支持搜索（按文件名、类型）和筛选
  - 支持拖拽素材到时间线或预览窗口
  - 显示素材缩略图、文件名、时长/尺寸信息
  - 右键素材：重命名 / 删除 / 在文件夹中显示 / 收藏
  - **空状态**：无素材时显示"拖拽文件到此处导入" + 虚线框提示
  - **加载状态**：导入中显示进度条和"导入中..."

#### 预设 Tab

- **预设库**：Remotion 官方预设 + 用户自定义预设
  - 顶部分类筛选栏（全部 / 数据图表 / 标题动画 / 转场 / 片头片尾 / 自定义）
  - 预设卡片网格布局（图标 + 名称 + 描述）
  - 点击预设 → 弹出确认对话框 → 应用到当前子项目（创建对应轨道和片段）
  - 右键自定义预设：编辑 / 重命名 / 删除
  - **空状态**：无自定义预设时显示"暂无自定义预设"

### 中间工作区

#### 预览窗口（上方）

```
┌────────────────────────────────────────┐
│ ◀ ▶ ⏸  00:00:00 / 00:00:10   [100%] 🔍 │  ← 工具栏
├────────────────────────────────────────┤
│                                        │
│         iframe（Remotion Player）      │
│                                        │
│                                        │
└────────────────────────────────────────┘
```

- **工具栏**：
  - 播放控制：跳到开头 ⏮ / 上一帧 ◀ / 播放/暂停 ⏸ / 下一帧 ▶ / 跳到结尾 ⏭
  - 时间码显示：`当前帧号 / 总帧数`（如 `150 / 300`）+ 换算时间 `00:05:00 / 00:10:00`
  - 缩放控制：下拉框（50% / 75% / 100% / 125% / 150% / 200%）
  - 全屏按钮 🔍（F11 快捷键）
- **iframe（Remotion Player）**：
  - 深色背景（与 Remotion 预览融合）
  - 点击 iframe 内元素 → 元素高亮（红色边框）→ 选中对应片段 → 右侧面板显示属性
  - 右键 iframe → 菜单：全屏 / 实际像素 / 适配窗口
- **空状态**：无子项目或 Generator 未生成时显示"等待生成..."
- **加载状态**：Generator 生成中显示半透明遮罩 + 旋转 loading 图标 + "正在生成预览..."
- **错误状态**：预览加载失败显示"预览加载失败" + "重新加载"按钮 + 错误详情展开

#### 对话面板（下方，可收起）

```
┌────────────────────────────────────────┐
│ 🤖 AI 正在生成动画...                    │  ← AI 思考状态
│ ┌──────────────────────────────────┐   │
│ │ 用户：创建一个标题写着 Hello      │   │  ← 用户消息
│ └──────────────────────────────────┘   │
│ ┌──────────────────────────────────┐   │
│ │ AI：好的，已为您创建标题动画。    │   │  ← AI 消息
│ │ [操作记录：创建了 TextLayer]      │   │
│ └──────────────────────────────────┘   │
│ ┌──────────────────────────────────┐   │
│ │ [图片预览] 参考图 1/3              │   │  ← 图片消息
│ └──────────────────────────────────┘   │
│ ┌──────────────────────────────────┐   │
│ │ ⚠️ 检测到安全违规，已阻止写入       │   │  ← 错误消息
│ └──────────────────────────────────┘   │
│                                        │
│ [请输入需求...              ] [🖼️] [➤] │  ← 输入区
└────────────────────────────────────────┘
```

**消息类型：**

| 类型 | 样式 | 内容 |
|------|------|------|
| **用户消息** | 右侧，蓝色气泡 | 文字内容 + 图片缩略图（可点击放大） |
| **AI 消息** | 左侧，灰色气泡 | 回复文字 + 操作记录折叠面板 |
| **系统消息** | 居中，浅灰色小字 | "已自动保存" / "生成完成" / "已撤销" |
| **错误消息** | 左侧，红色气泡 | 错误说明 + 重试按钮 |
| **AI 思考中** | 顶部浮动条 | 旋转 loading + "AI 正在理解您的需求..." |

**输入区：**
- 多行文本输入框（自动增高，最大 5 行）
- 图片上传按钮 🖼️（点击选择文件或粘贴图片）
- 发送按钮 ➤（Enter 发送，Shift+Enter 换行）
- 快捷命令：输入 `/` 弹出命令列表（`/preset` / `/undo` / `/redo`）

**交互行为：**
- **可收起**：点击顶部 ▼ 收起为单行输入框，最大化预览区域
- **展开**：点击输入框或 ▲ 恢复到完整高度
- **滚动**：消息过多时自动滚动到底部，用户手动向上滚动时暂停自动滚动
- **清空**：支持清空当前子项目对话历史（保留 project.json 中的记录）
- **右键消息**：复制 / 删除单条消息

#### 状态 A：选中元素时 → 显示属性面板

当用户在时间线或预览窗口选中某个片段/轨道时，右侧面板显示该元素的完整属性编辑界面。

**信息栏（顶部固定）：**
- 片段 ID + 名称（可双击重命名）
- 片段类型标签（如"文字片段"、"图片片段"）
- 片段在时间线上的起止帧范围（如"0 → 90 帧"）

**属性分组（可折叠）：**

- **变换**（默认展开）：
  - 位置：X/Y 坐标输入框（支持直接输入像素值），输入时预览窗口实时同步
  - 缩放：百分比滑块（10%-500%）+ 数值输入框
  - 旋转：角度输入框（-360°~360°）+ 圆形旋转控件（鼠标拖拽旋转）
  - 透明度：0-100% 滑块 + 数值输入框
- **动画**（默认展开）：
  - 入场动画：类型下拉框（淡入 / 从左滑入 / 从右滑入 / 缩放弹出 / 打字机 / 无） + 持续时间输入框（帧数）
  - 出场动画：同上
  - 缓动函数：下拉框（linear / ease-in / ease-out / ease-in-out / spring）
- **样式**（根据片段类型动态显示）：
  - **文字类**：字体下拉框（系统字体 + 项目字体）、字号输入、颜色选择器（支持吸管工具吸取预览窗口颜色）、行高、字间距、对齐方式（左/中/右）
  - **图片类**：填充模式（cover / contain / fill）、圆角输入、滤镜下拉框（亮度/对比度/饱和度/模糊）
  - **形状类**：填充色选择器、描边色选择器、描边宽度输入
  - **视频类**：音量滑块（0-100%）、播放速度（0.25x-2x）
- **关键帧**（默认折叠，有关键帧时自动展开）：
  - 当前属性的关键帧列表（时间 + 值 + 缓动）
  - 添加/删除/编辑关键帧按钮
  - 缓动曲线可视化（小型 SVG 曲线图）
  - 双击时间值可跳转时间线到该帧
- **数据绑定**（仅数据动画片段显示）：
  - 数据文件选择下拉框（项目 data/ 目录中的 CSV/JSON）
  - 字段映射下拉框（自动读取表头）
  - 数据预览表格（前 5 行）

**快捷操作栏（底部固定）：**
- [复制片段] [删除片段] [重置为预设] 按钮

#### 状态 B：未选中元素时 → 显示快捷素材区

当没有元素被选中时，右侧面板切换为快捷操作区。

- **快捷素材区**：
  - 显示**最近使用的 10 个素材**（跨子项目，按 `last_used_at` 排序）
  - 显示**当前子项目**的素材（与左侧面板主素材库联动，但只显示当前子项目相关）
  - 素材以缩略图网格形式展示（2 列），方便快速拖拽到时间线
  - 支持"收藏"标记 ⭐，常用素材置顶
- **快捷操作**：
  - "+ 添加文字"按钮 → 在时间线当前位置快速创建 text 轨道和片段
  - "+ 添加图片"按钮 → 打开素材选择对话框
  - "+ 添加形状"按钮 → 弹出形状类型选择（矩形/圆形/线条）→ 快速创建 shape 片段
  - "+ 添加图表"按钮 → 弹出图表类型选择（柱状图/折线图/饼图）
- **最近预设**：
  - 显示最近使用的 5 个预设，点击直接应用

**面板切换逻辑**：
- 选中元素 → 自动切换为属性面板（带 200ms 淡入动画）
- 取消选中（点击空白处或按 Esc）→ 自动切换为快捷素材区
- 切换时保持滚动位置（属性面板和快捷素材区独立记忆滚动位置）

**属性编辑的实时反馈：**
- 数值输入框：失去焦点或按 Enter 时触发 Generator（防抖 500ms）
- 滑块：拖拽过程中实时更新 previewStore，拖拽结束后触发 Generator
- 颜色选择器：选择颜色后立即更新预览（无需防抖，颜色变更轻量）
- 下拉框：选择后立即触发 Generator

## 底部时间线

```
┌──────────────────────────────────────────────────────────────────┐
│ ➕ │🔒│👁️│🔇│ [轨道1 文字]  ━━━[片段A]━━━━━━━[片段B]━━━ │ ◆ │
│ ➕ │🔒│👁️│🔇│ [轨道2 图片]  ━━━━━[片段C]━━━━━━━━━━━━━━ │ ◆ │
│ ➕ │🔒│👁️│🔇│ [轨道3 音频]  ━━━━━━━━━[片段D]━━━━━━━ │ ◆ │
├──────────────────────────────────────────────────────────────────┤
│ ◀──▶ [播放头]  00:00:05 / 00:00:10    [1s▼] [===▓====]    │
└──────────────────────────────────────────────────────────────────┘
       ↑轨道控制区     ↑轨道内容区                  ↑播放控制区
```

### 时间线区域划分

| 区域 | 宽度 | 说明 |
|------|------|------|
| **轨道控制区** | 200px（固定） | 轨道操作按钮 + 轨道名称 + 轨道图标 |
| **轨道内容区** | 自适应（剩余宽度） | 片段块 + 播放头 + 关键帧标记 |
| **播放控制区** | 80px（固定） | 轨道折叠/展开、轨道设置 |

### 轨道控制区

**每行轨道包含：**
- **展开/折叠** ▼/▶：折叠后只显示轨道名称栏，隐藏片段内容
- **锁定** 🔒/🔓：锁定后禁止编辑该轨道的片段
- **可见** 👁️/🙈：隐藏/显示该轨道在预览中
- **静音** 🔇/🔊（仅音频轨道显示）
- **轨道名称**：可双击重命名，右键菜单（重命名 / 删除 / 复制 / 颜色标记）
- **轨道类型图标**：文字 📝 / 图片 🖼️ / 视频 🎬 / 音频 🔊 / 形状 ⬜ / 图表 📊
- **添加片段** ➕：点击在轨道当前位置添加默认片段

### 轨道内容区

**片段显示：**
- 矩形块，宽度 = 持续时间 × 缩放比例
- 块内显示：片段名称（截断显示...）+ 类型图标
- 颜色编码：文字=蓝色 / 图片=绿色 / 视频=紫色 / 音频=橙色 / 形状=红色 / 图表=青色
- 片段边框：未选中=灰色细线，选中=蓝色粗线 + 阴影
- 片段左右边缘可拖拽调整时长（鼠标变为 ↔️）
- 片段主体可拖拽调整位置（鼠标变为 ✋）

**关键帧标记：**
- 片段上方/下方显示菱形标记 ◆，颜色与片段一致
- 悬停菱形标记显示关键帧详情（属性 + 值 + 帧号）
- 点击菱形标记跳转时间线到该帧 + 选中该属性

**播放头：**
- 垂直红色细线，贯穿所有轨道
- 顶部显示倒三角 ▼ 指示器（可拖拽）
- 拖拽时显示当前帧号 tooltip
- 释放时预览窗口 seek 到该帧

**时间标尺：**
- 轨道内容区顶部显示时间标尺（帧号 / 秒 / 分钟）
- 刻度随缩放级别动态调整（放大显示帧，缩小显示秒）

### 播放控制区（底部）

- **播放控制**：跳到开头 ⏮ / 上一帧 ◀ / 播放/暂停 ⏸ / 下一帧 ▶ / 跳到结尾 ⏭
- **时间码**：`当前帧号 / 总帧数`（如 `150 / 300`）+ 换算时间
- **缩放控制**：下拉框（适应 / 帧 / 秒 / 分钟）+ 滑块（10%-500%）
- **循环播放** 🔁：开关按钮
- **吸附开关** 🧲：启用/禁用吸附

### 时间线交互手势

| 手势 | 作用区域 | 行为 |
|------|----------|------|
| **单击片段** | 轨道内容区 | 选中片段，右侧面板显示属性 |
| **双击片段** | 轨道内容区 | 选中片段 + 预览窗口 seek 到片段起始帧 |
| **拖拽片段** | 轨道内容区 | 移动片段位置（修改 startInFrames）|
| **拖拽片段边缘** | 片段左右边缘 | 调整片段时长（修改 durationInFrames）|
| **拖拽播放头** | 时间标尺 | 改变当前播放位置 |
| **拖拽轨道** | 轨道控制区 | 调整轨道顺序（修改 order）|
| **右键片段** | 轨道内容区 | 菜单：复制 / 粘贴 / 删除 / 拆分 / 添加关键帧 |
| **右键轨道** | 轨道控制区 | 菜单：重命名 / 删除 / 复制 / 颜色 / 锁定 / 隐藏 |
| **滚轮** | 轨道内容区 | 水平滚动时间线（Shift+滚轮）/ 缩放（Ctrl+滚轮）|
| **空格** | 全局 | 播放/暂停 |
| **Delete** | 选中片段时 | 删除片段 |
| **Ctrl+Z** | 全局 | 撤销 |
| **Ctrl+Y** | 全局 | 重做 |

### 时间线空状态

- **无轨道**：显示"暂无轨道，点击 + 添加轨道或让 AI 生成动画"
- **轨道无片段**：显示虚线占位 "拖拽素材到此处" 或 "点击 + 添加片段"

### 响应式适配策略

**最小窗口尺寸**：1200×700（低于此尺寸时显示提示"窗口过小，建议最大化使用"）

**不同分辨率下的布局策略**：

| 窗口宽度 | 布局调整 |
|----------|----------|
| ≥ 1600px | 三栏完整显示，所有面板默认展开 |
| 1200-1599px | 左侧面板默认 220px，右侧面板默认 260px |
| < 1400px | 对话面板默认收起（只显示输入框），预览区域最大化 |
| < 1200px | 触发最小尺寸警告 |

**高 DPI 适配**：
- 支持 100%、125%、150%、175%、200% 系统缩放
- UI 元素使用相对单位（rem/em），文字和按钮随 DPI 自动缩放
- 图标使用 SVG，保证任意缩放清晰度
