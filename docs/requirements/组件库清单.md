本文档列出 EasyMotion 项目中使用的所有 UI 组件，包括 shadcn/ui 内置组件、自定义业务组件和组件依赖关系。作为前端开发的组件速查手册。

> **最后更新**：2026-05-30  
> **关联文档**：UI布局设计.md、应用架构详细设计.md、项目编码规范.md

    --foreground: 210 40% 98%;        /* #F8FAFC */
    --card: 240 10% 3.9%;
    --card-foreground: 210 40% 98%;
    --popover: 240 10% 3.9%;
    --popover-foreground: 210 40% 98%;
    --primary: 347 77% 50%;           /* #E11D48 */
    --primary-foreground: 0 0% 100%;
    --secondary: 240 20% 15%;         /* #1E1B4B */
    --secondary-foreground: 210 40% 98%;
    --muted: 240 20% 15%;
    --muted-foreground: 215 20% 65%; /* #94A3B8 */
    --accent: 240 20% 15%;
    --accent-foreground: 210 40% 98%;
    --destructive: 0 84% 60%;         /* #DC2626 */
    --destructive-foreground: 0 0% 100%;
    --border: 240 15% 25%;            /* #2D2D4A */
    --input: 240 15% 25%;
    --ring: 347 77% 50%;              /* #E11D48 */

## 4. 组件开发规范

### 4.1 目录结构

```
src/components/
├── layout/           # 布局组件
├── timeline/         # 时间线组件
├── preview/          # 预览组件
├── conversation/     # 对话组件
├── property-panel/   # 属性面板组件
└── common/           # 通用组件

每个组件目录：
├── ComponentName.tsx       # 组件实现
├── ComponentName.test.tsx  # 单元测试
├── index.ts                # 导出
└── types.ts                # 组件专属类型（可选）
```

### 4.2 Props 命名规范

```typescript
interface ComponentNameProps {
  // 必填在前，选填在后
  requiredProp: string;
  optionalProp?: number;
  
  // 回调以 on 开头
  onClick?: () => void;
  onChange?: (value: string) => void;
  
  // 子组件传递以 render 开头
  renderItem?: (item: Item) => React.ReactNode;
  
  // 样式覆盖
  className?: string;
}
```

### 4.3 可访问性要求

每个组件必须：
- 可聚焦元素有 `aria-label` 或可见文本。
- 支持 `ref` 转发（使用 `React.forwardRef`）。
- 颜色信息有图标/文字冗余。
- 动画支持 `prefers-reduced-motion`。

### 4.4 性能要求

- 列表组件使用 `React.memo` + `useMemo` 避免不必要重渲染。
- 滚动区域使用虚拟化（`react-window`）当项目数 > 100。
- 避免在 `render` 中创建新对象/函数。

## 5. 图标库

使用 **Lucide React** 作为图标库，统一尺寸：

| 场景 | 尺寸 | 示例 |
|------|------|------|
| 工具栏图标 | 16px (`w-4 h-4`) | 播放、暂停、撤销 |
| 按钮图标 | 16-20px | 发送、添加、删除 |
| 面板图标 | 20px (`w-5 h-5`) | 轨道类型、素材类型 |
| 空状态图标 | 48px (`w-12 h-12`) | 空素材库、无项目 |
| 菜单图标 | 16px | 文件、编辑菜单项 |

**禁止**：使用 Emoji 作为 UI 图标。

## 6. 动画组件

| 组件 | 库 | 用途 |
|------|-----|------|
| 面板展开/收起 | CSS `transition` | 左/右/底部面板 |
| 弹窗出现/消失 | `framer-motion` | Dialog、Sheet |
| 列表项进入 | `framer-motion` | 消息列表、素材列表 |
| 拖拽反馈 | `@dnd-kit` | 时间线片段拖拽、素材拖拽 |
| 播放头移动 | 无动画（实时） | 精确到帧 |
| 进度条 | CSS `transition` | 导入、导出、生成 |
