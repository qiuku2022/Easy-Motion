## 依赖顺序

```
remotion-sandbox → remotion-context → remotion-code tools
                    ↓
              compile-check
                    ↓
         custom-registry 模板 + MainSequence patch
                    ↓
         registerCustomComponent + agent-service 预览
                    ↓
              eval + 文档回写
```
