## 路径模型

用户打开项目后，Agent 上下文已有：

```javascript

  projectPath,      // 例如 D:\Videos\MyIntro.em
```

```
{projectPath}/{subprojectPath}/remotion/
├── package.json
├── public/
└── src/

    ├── Root.tsx               🔒 只读（M5.2）

    ├── preview-entry.tsx      🔒 只读

    │   ├── MainSequence.tsx   🔒 只读（timeline 驱动，Agent 不改）
    │   ├── layers/            🔒 只读（标准 Layer，仅 import）
    │   ├── newsletter-design/ ⚠️ 只读（内置设计组件）
    │   └── custom/            ✅ Agent 可读写（主沙箱）
    ├── presets/
    │   ├── registry.ts        🔒 只读（vendor 自动生成）

    └── lib/                   🔒 只读
```

图例：🔒 Agent 不可写 | ✅ Agent 可写 | ⚠️ 第一版只读，后续可开放

## 沙箱实现要点

新建 `apps/electron/src/main/agent/remotion-sandbox.js`：

```javascript
function resolveRemotionPaths(projectPath, subprojectPath) {
  const remotionDir = path.join(projectPath, subprojectPath, "remotion");
  const srcDir = path.join(remotionDir, "src");
  return { remotionDir, srcDir };
}

function assertWritableRelativePath(relativePath) {
  const normalized = path.normalize(relativePath).replace(/^(\.\.(\/|\\|$))+/, "");
  if (normalized.includes("..")) throw new Error("E2410: 路径越界");
  const allowedPrefixes = [
    "components/custom/",
    "presets/custom-registry.ts",
  ];
  if (!allowedPrefixes.some((p) => normalized === p || normalized.startsWith(p))) {
    throw new Error(`E2410: 不允许写入 ${relativePath}`);
  }
  return normalized;
}
```

**读取**可放宽到整个 `src/`（便于 Agent 理解现有结构），**写入**严格限定上表。

## 自定义组件如何进预览

现有 `MainSequence.tsx` 通过 `resolvePresetComponent(clip.source.component)` 解析 animation 片段。

**方案（推荐）**：不修改 `registry.ts`，新增用户侧注册表：

1. 模板新增 `src/presets/custom-registry.ts`（新建项目时复制；旧项目 `ensureCustomRegistry` 迁移）
2. `custom-registry.ts` 导出 `CUSTOM_COMPONENT_MAP` + `resolveCustomComponent(name)`
3. **一次性** patch 用户 `MainSequence.tsx`（或通过 `preview-service.ensureCustomComponentSupport`）：

   ```tsx
   import { resolveCustomComponent } from "../presets/custom-registry";
   // 在 resolvePresetComponent 失败时：
   const Custom = resolveCustomComponent(clip.source.component);
   ```

4. Agent 工具 `registerCustomComponent`：
   - 写 `components/custom/MyIntro.tsx`
   - 更新 `custom-registry.ts` 增加 import + map 条目
   - 调用现有 `applyPreset` 等价逻辑，或 `createClip` animation 轨道，`source.component = "MyIntro"`

## 与 Generator / 漂移检测

| 场景 | 行为 |
|------|------|
| timeline 驱动预览（含 flattenClipsForPreview） | 改 custom TSX → HMR；改 timeline → TIMELINE_UPDATE |
| 非 timeline 驱动 + Generator 路径 | Agent 改 custom/ 不触发 Generator；勿改 Generator 输出物 |
| `checkRemotionDrift` | 写 custom/ 会改变 fingerprint；**排除** `components/custom/**` 与 `custom-registry.ts` 的 drift 告警（或标记 `remotionSyncSource: "agent-code"`) |

## 写入快照与回滚

`RemotionContext` 维护：

```javascript
this.fileSnapshots = new Map(); // relativePath → content before write
this.remotionChanged = false;
this.changeLog = []; // { op, path, ... }
```

单次对话内 compile 失败 → 从 snapshot 恢复；成功 → 清空 snapshot。
