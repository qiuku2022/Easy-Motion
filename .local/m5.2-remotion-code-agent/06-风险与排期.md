# 06 — 风险与排期

## 风险

| 风险 | 影响 | 缓解 |
|------|------|------|
| 用户 remotion 项目无 `node_modules` | compile check 失败 | 打开项目/预览前 `ensureRemotionDeps`；降级为 security-scan + import 白名单 |
| Agent import 未安装包（如 @remotion/shapes） | Vite 编译报错 | 读用户 package.json 做 import 白名单 |
| MainSequence 未 patch custom 支持 | 组件写了但预览不显示 | `ensureCustomComponentSupport` 打开项目时 idempotent 迁移 |
| LLM 不写 custom 只聊天 | 与 M5 相同体感 | compile 失败硬反馈；prompt + retry；eval 回归 |
| 写 TSX 触发 drift 误报 | UI 提示「从 Remotion 同步」 | drift 排除 `components/custom/**` |
| Windows spawn tsc 路径问题 | compile 超时 | 与 preview-service 相同 cmd.exe 包装 |
| 旧项目无 custom-registry | register 失败 | `ensureCustomRegistry` 自动创建 |
| Token 消耗增大 | 读大文件占 context | readRemotionFile 默认 limit 200 行；searchRemotionDocs 截断 |
