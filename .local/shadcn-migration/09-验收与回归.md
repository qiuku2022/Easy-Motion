## 1. 视觉验收

### 1.1 主题

- [ ] 主背景 `#0F0F23` 感知正确（`bg-background`）
- [ ] 主强调色为 Action Red（`bg-primary`），非默认 zinc 白 primary
- [ ] 输入框聚焦为 Teal 环（`--ring`）
- [ ] 侧栏、时间线、预览三区边框一致（`border-border`）

### 1.2 字体

- [ ] UI 文字为 Plus Jakarta Sans
- [ ] 时间码为 JetBrains Mono

- [ ] 所有主按钮 `rounded-sm`（4px）
- [ ] 面板圆角 `rounded-lg`（8px）与文档一致
- [ ] 图标统一 lucide，`h-4 w-4` 工具栏尺寸

## 3. 无障碍验收

依据 [可访问性设计](../../docs/requirements/可访问性设计.md) WCAG 2.1 AA：

- [ ] Tab 键可遍历工具栏、Tab 面板、表单字段
- [ ] 焦点环可见（`ring-ring`）
- [ ] 图标按钮有 `sr-only` 文本或 `Tooltip`
- [ ] 对话框（如有）含 `DialogTitle`
- [ ] 颜色不是唯一状态指示（轨道锁定等仍有图标）
- [ ] `prefers-reduced-motion: reduce` 下动画减少

### 键盘

- [ ] Escape 关闭 Dropdown / ContextMenu / Dialog
- [ ] 时间线快捷键（`useTimelineShortcuts`）仍有效

### Pre-Delivery Checklist（技能内置）

**Visual**

- [ ] 无 emoji 作图标
- [ ] 图标集一致（Lucide）
- [ ] Hover 不引发布局位移
- [ ] 使用 `bg-primary` 等语义色，非随意 hex

**Interaction**

- [ ] 可点击元素 `cursor-pointer`（`Button` 默认已含）
- [ ] 过渡 150–300ms

**Layout**

- [ ] 无内容被固定栏遮挡
- [ ] 无意外横向滚动
