## 3. 官方表单模式（必须遵循）

依据 [Form 文档](https://ui.shadcn.com/docs/components/form)：

```tsx
const formSchema = z.object({
  name: z.string().min(1, "名称不能为空"),
  opacity: z.number().min(0).max(100),
});

const form = useForm<z.infer<typeof formSchema>>({
  resolver: zodResolver(formSchema),
  defaultValues: { name: "", opacity: 100 },
});

<Form {...form}>
  <form onSubmit={form.handleSubmit(onSubmit)}>
    <FormField
      control={form.control}
      name="name"
      render={({ field }) => (
        <FormItem>
          <FormLabel>名称</FormLabel>
          <FormControl>
            <Input {...field} />
          </FormControl>
          <FormMessage />
        </FormItem>
      )}
    />
  </form>
</Form>
```

### 准则

- `FormField` + `FormItem` + `FormLabel` + `FormControl` + `FormMessage`（[ui-ux-pro-max shadcn 准则](https://ui.shadcn.com/docs/components/form)）
- 校验用 Zod，避免手写 `if (!value)` 分散逻辑
- 错误展示用 `FormMessage`，不用裸 `<span className="text-red-500">`

## 4. 与 Zustand 的集成

当前属性来自 `timelineStore` 选中 clip。推荐模式：

```
选中 clip 变化 → form.reset(clipToFormValues(clip))
字段 onChange → debounce → store.updateClip(...)
提交（如有）→ form.handleSubmit → store + IPC 持久化
```

**注意**：

- 避免 `useForm` 与 store 双写冲突；以 **store 为 source of truth**，form 为编辑缓冲
- `useEffect` 监听 `selectedClipId` 调用 `form.reset`

## 5. 字段类型映射

`ClipPropertyFields.tsx` 中 `field.type` → shadcn 组件：

| type | 组件 |
|------|------|
| `text` | `Input` |
| `number` | `Input type="number"` |
| `textarea` | `Textarea` |
| `boolean` | `Switch` |
| `select` | `Select` + `SelectItem` |
| `color` | 暂保留自定义 ColorPicker；Popover 包裹（阶段 5b） |
| `range` | `Slider` |

### Slider 与关键帧

音量、透明度等：`Slider` + `onValueCommit` 写 store，拖拽中可用 `onValueChange` 本地 state 减少 store 更新频率。

## 6. PropertiesPanel 结构

```
PropertiesPanel
├── Tabs（阶段 3 已迁移）
├── ScrollArea（阶段 4）
└── Form
    ├── 变换 / 样式 / 文本 分组
    │   └── FormField × N
    └── 空状态（未选中）→ 文案 + 可选 Card
```

空状态可用：

```tsx
<div className="flex flex-col items-center justify-center p-6 text-muted-foreground">
  <p className="text-sm">选中时间线上的片段以编辑属性</p>
</div>
```
