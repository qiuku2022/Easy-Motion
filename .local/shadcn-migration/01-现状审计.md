## 4. 样式使用统计

当前约 **25 个** 组件文件使用 `em-*` 或 `cn()`。高频 class：

- `bg-em-bg`, `bg-em-surface`, `bg-em-elevated`
- `border-em-border`, `text-em-text`, `text-em-muted`
- `text-em-teal`, `bg-em-accent`（强调色）

`index.css` 已定义 shadcn 风格变量（HSL 无 `hsl()` 包装格式），但 `tailwind.config.js` **未映射** `background`/`primary` 等语义色，导致组件未使用语义类。

## 7. 审计结论：推荐迁移路径

1. **不升级 Tailwind v4**（除非单独立项）；在 v3 + PostCSS 上跑 shadcn CLI
2. **先主题后组件**；避免装完组件再改 CSS 导致大量覆盖
3. **三个样板**验证模式：`PanelTabs` → `AddTrackMenu` → `ContextMenu`
4. **时间线核心不动**，只换工具栏/菜单/表单 primitive
