# 02 — 推荐方案（拍板用）

## 推荐：**方案 A — shadcn Neutral Dark**

### 理由

1. 你明确接受「shadcn 默认黑白主题」— 与 `components.json` 的 `baseColor: neutral` 一致，**纠正**当前 CSS 与配置脱节。  
2. 改动集中在 `index.css` + 按钮用法，风险低、可快速看到效果。  
3. 专业工具趋势是 **低饱和壳层**；红色留给 destructive，而不是「保存项目」。  
4. ui-ux-pro-max 对「视频编辑器」的自动推荐仍指向旧 Cinema 色 — **不应**再跟随该条 palette。

### 拍板后 UI 行为约定

| 场景 | variant | 说明 |
|------|---------|------|
| 顶栏图标 | `ghost` | 已大部分如此，保持 |
| 保存 / 新建 / 导入 | `outline` 或 `secondary` | **不再** `default` |
| 唯一强 CTA（如「生成预览」） | `default` | 白按钮，克制使用 |
| 删除 | `destructive` | 浅红底，非大红块 |
| AI 发送 | `default` size `icon` | 白图标按钮即可 |

### 圆角

- **删除**业务组件上多余的 `rounded-sm`（让 nova `rounded-lg` / 尺寸 token 生效）  
- 时间线内小控件（轨道头图标 `h-6`）可保留 `rounded-sm` 或 `size="icon-xs"`  
- 全局 `--radius: 0.625rem`（10px）— shadcn 2025 默认，比旧文档 8px 更现代

### 字体

**保留** Plus Jakarta Sans + JetBrains Mono（无需为「现代」强行换 Archivo/Space Grotesk）。
