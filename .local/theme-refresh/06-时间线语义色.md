# 06 — 时间线语义色（方案 B 专用；A 可暂不改画布）

时间线画布（`ClipBlock`、`Playhead`、`TimelineRuler`）当前大量硬编码 `em-teal` / 红色选中。  
**壳层换肤后**，若播放头仍是青色、按钮却是白色，会不协调。

---

## 策略

| 模式 | 时间线 |
|------|--------|
| 方案 A | 可第二阶段再统一；短期允许画布保留旧 accent |
| 方案 B | 与壳层同时改，引入 `--timeline-*` |

---

## 建议 token（方案 B）

| Token | 用途 | 组件 |
|-------|------|------|
| `--timeline-accent` | 播放头、选中边、可见轨道图标 | `Playhead`, `ClipBlock`, `TrackHeader` Icon |
| `--timeline-accent-muted` | 选中片段填充 | `ClipBlock` |
| `--timeline-snap-guide` | 吸附线 | `SnapGuides` |
| `--timeline-ruler-*` | 刻度 | `TimelineRuler` |

---

## 组件改动要点

```
Playhead.tsx          → bg-timeline-playhead（无 transition）
ClipBlock.tsx         → 选中 border/background 用 timeline token
TrackHeader.tsx       → 可见轨道图标 text-timeline-accent
SnapGuides.tsx        → stroke timeline-snap-guide
TimelineRuler.tsx     → 刻度线用 muted gray，非紫边
```

**不要**用 `bg-primary` 画播放头（方案 A 下 primary 是白色按钮色）。

---

## 方案 A 最小改动（可选）

若只想先改壳层、暂不动画布：

- 接受短期「白按钮 + 青绿播放头」共存  
- 在 README 注明为过渡态  
- 第二阶段再执行本文件
