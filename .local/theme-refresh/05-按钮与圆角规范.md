## 1. Variant 分层

### 壳层（顶栏 / 面板 / 对话框）

| 优先级 | variant | 示例 |
|--------|---------|------|
| 默认 | `ghost` | 撤销、缩放、轨道图标 |
| 次要操作 | `outline` | 打开项目、添加轨道、重试预览 |
| 填充次要 | `secondary` | 批量操作、次要确认 |
| 高强调（稀少） | `default` | 全应用 1–2 个主 CTA |
| 危险 | `destructive` | 删除片段（配合 AlertDialog） |
| 链接式 | `link` | 文案链接 |

### 禁止模式

```tsx
// ❌ 保存/新建不要用 default（在旧主题下 = 满屏红）
<Button variant="default">保存</Button>

// ✅
<Button variant="outline">保存</Button>
```

## 2. 圆角：跟 nova，别跟旧文档 §0.4

| 元素 | 旧文档 | 新规范（nova） |
|------|--------|----------------|
| Button | `rounded-sm` 4px | **默认** `rounded-lg`（来自 cva） |
| Input / Select | `rounded-md` | 组件自带，勿覆写 |
| 面板 | `rounded-lg` 8px | `rounded-lg`（与 `--radius` 一致） |
| 时间线轨道头 icon | 可 `size="icon-xs"` | 用 size token，非手写 `rounded-sm` |

### 清理清单（实施时 grep）

```bash
# 业务层不必要的 rounded-sm（保留时间线画布内例外）
rg 'rounded-sm' apps/electron/src/renderer/src/components --glob '*.tsx'
```

原则：**除非设计理由明确，删除 Button 上的 `className="... rounded-sm"`**。

## 4. 聚焦环

## 5. 动效

保留：

- `transition-all` / 150ms（nova button 默认）
- `prefers-reduced-motion`（已在 index.css）

可选弱化（若觉得「廉价」）：

- nova 的 `active:translate-y-px` — 可在实施时从 `button.tsx` 去掉，**仅当**你确认需要；优先不动组件库。
